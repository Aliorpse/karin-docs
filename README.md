# karin文档

```bash
git clone -b docs https://github.com/KarinJS/karin-docs.git
```

## 安装依赖

```bash
cd karin-docs && pnpm install
```

## 启动项目

```bash
pnpm dev
```

## 编译

```bash
pnpm build
```

## 预览

```bash
pnpm preview
```
