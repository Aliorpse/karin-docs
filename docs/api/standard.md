## 调用方法

- 在插件内部调用

::: code-group

```js [被动消息]
const res = await this.e.bot.GetVersion()
console.log(res)

```

```js [主动调用] twoslash
import { Bot } from 'node-karin'

const self_id = 123456789

// 获取机器人实例
const bot = Bot.getBot(self_id)
// 需要注意 如果ID对应的机器人实例不存在会返回undefined
const res = await bot.GetVersion()
console.log(res)

```
:::
